## bizlp-website

Next.js + TypeScript + Tailwind CSS example with microCMS integration.

### Quick Start

1. Install dependencies

```bash
npm install
npm install @tailwindcss/line-clamp --save
```

2. Create `.env.local`

```bash
MICROCMS_SERVICE_DOMAIN=bizlp-website
MICROCMS_API_KEY=VHpIA2GF0K1Srz3p8rlGdmxiyrJxSPPW9dSB
```

3. Run dev server

```bash
npm run dev
```

Open http://localhost:3000
